Método EAMGA

Enseñanza Aprendizaje de Matemáticas Guiadas por Algoritmos
Para el estudiante:
Para el Estudiante (Método CPR3):
1. Comprender el Problema: Reconoce el tipo de problema y el método adecuado.
  • Lee detalladamente y entiende el problema
  • Determina el método o algoritmo a utilizar
  • Determina las Entradas relevantes del problema
  • Determina las Salidas que se producirá
2. Planear: Con los datos anteriores determina cómo vas a resolver el problema, cómo va a utilizar los datos de entrada y qué tipo de resultado puedes obtener.
3. Resolver aplicando el Algoritmo Paso a Paso: Sigue los pasos del método de manera ordenada.
4. Revisar los Cálculos: Revisa cada paso para asegurar precisión.
  • Si es posible emplea los métodos de comprobación del algoritmo.
  • Interpreta el resultado verificando si es lógico de acuerdo a las entradas.
  • ESCRIBE claramente la solución final.
6. Reflexionar sobre el Proceso: Evalúa el proceso y considera posibles mejoras.
